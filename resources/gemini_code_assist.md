# Installing Gemini Code Assist
## Creating with AI in the Loop â€” VS Code extension setup

Gemini Code Assist is an AI coding assistant from Google that integrates directly into VS Code. It provides code completions, chat-based help, and code generation.

---

## What you need

- VS Code installed
- A personal Google account (not your .edu account)

---

## Step-by-step: Install and sign in

1. Open VS Code.

2. Go to the Extensions view (`Ctrl+Shift+X` on Windows, `Cmd+Shift+X` on macOS).

3. Search for **Gemini Code Assist**. Look for the one by **Google** with a verified badge.

4. Click **Install**. Restart VS Code if prompted.

5. Click the **Gemini icon** in the left sidebar (a pinched diamond shape).

6. Click **Login to Google** and sign in with your **personal Google account** (not your .edu account).

7. If prompted to open an external website, click **Open**.

8. Review and accept the privacy notice that appears in the chat panel.

---

## Verify it's working

After signing in, you should see the Gemini chat panel. Try typing a question like "How do I read a file in Python?" to confirm it's working.

---

## Free tier

- Gemini Code Assist can be used for free with a free tier Google Account
- If you hit usage limits, you may need to wait until the next day for limits to reset
- There is also a paid tier with higher limits if needed

---

# Troubleshooting

## Can't sign in / authentication errors

- Make sure you're using a **personal** Google account, not your .edu account
- Try signing out and back in
- Try a different browser for the authentication flow

## Extension not appearing in sidebar

- Restart VS Code after installation
- Check that the extension is enabled in the Extensions view

## "Usage limit exceeded"

- Wait until the next day for limits to reset
